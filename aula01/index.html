HTML e CSS

Definição

HTML (HyperText Markup Language) é uma linguagem de marcação utilizada para estruturar o conteudo da página web que você visita.

CSS ( Planilhas de estilo em cascata) é usada para estilizar páginas web por exemplo, para alterar a fonte, cor, tamanho e espaçamento do seu conteúdo, ou seja responsável pelo estilo da página.

Web Site

Podemos considerar um Web site uma coleção de páginas HTML estáticas, ou seja, que não interagem com um banco de dados através de uma linguagem de servidor Web.

Aplicação Web

Uma aplicação Web pode conter uma coleção de páginas, porém o conteúdo destas páginas é carregado através de solicitações (requisições) à um banco de dados, que conterá armazenado os textos, imagens ou mídias que a página precisa exibir.
